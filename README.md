# Projeto Bootcamp Dio e Blip

## Projeto classificador de heróis 
## Professor Felipe Aguiar “Felipão”
## Aluna Kamille Rodrigues de Oliveira Almeida

- O projeto envolve: Variáveis, operadores, laços de repetição e estruturas de decisões
O objetivo é classificar um herói em diferentes níveis de acordo com a quantidade de xp que ele possui.

Se XP for menor do que 1.000 = Ferro

Se XP for entre 1.001 e 2.000 = Bronze

Se XP for entre 2.001 e 5.000 = Prata

Se XP for entre 5.001 e 7.000 = Ouro

Se XP for entre 7.001 e 8.000 = Platina

Se XP for entre 8.001 e 9.000 = Ascendente

Se XP for entre 9.001 e 10.000= Imortal

Se XP for maior ou igual a 10.001 = Radiante

- Ao final a mensagem "O Herói de nome *{nome}* está no nível de *{nivel}*" deve aparecer.

- Para essa tarefa as variáveis nome, xp e nivel foram criadas. A variável nivel permaneceu vazia. Após isso foram usados if e if else para definir a variável nivel, com base no xp, de acordo com a tabela acima, e a mensagem que irá aparecer.
